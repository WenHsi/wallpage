# 壁紙工作室

**在手機桌布上寫下你的句子。**

免費線上壁紙產生器，輸入文字、挑選字體與配色，一鍵下載專屬手機鎖定畫面桌布。
無需安裝、無需登入，開啟即用。

🔗 **[立即使用 →](https://wenhsi.github.io/wallpaper-studio/)**

---

## 功能

### 文字
- 自由輸入多行文字，支援中英文
- 14 款字體（思源宋體、霞鶩文楷、Playfair Display 等）
- 字體大小、行距、字距、對齊方式調整
- 文字顏色、透明度、陰影、文字背景色塊

### 背景
- **單色**：內建暖色系色票，支援自訂顏色
- **漸層**：多組預設漸層，可自訂起終色與方向
- **色塊漸層（Mesh）**：4 色混合的柔和漸層
- **噪點質感**：帶有紙紋顆粒感的底色
- **圖片**：上傳照片，可調整位置、縮放、模糊、遮色

### 配色組合
- 儲存「文字色 + 背景色」為一組配色，點擊一鍵套用
- 支援星號固定，清除時保留星號配色

### 排版
- 智能排版：自動避開鎖定畫面的時鐘、小工具與工具列
- 支援直向與橫向預覽
- 手動排版：直接拖動文字到任意位置

### 輸出
- 依機型輸出對應解析度圖片
- 支援分享連結（含所有設定，不含圖片）
- iOS 裝置可直接透過分享存入相簿

### 支援機型
| 系列 | 型號 |
|------|------|
| iPhone | SE 3、13 / 14、13 Pro Max / 14 Plus、14 Pro / 15 / 16 系列 |
| Samsung | S24、S24 Ultra / S25 系列 |
| Pixel | 8 / 9 |
| 通用 | 1080p（1920×1080）|
| 此裝置 | 自動偵測當前螢幕尺寸 |

---

## 其他
- 歷史紀錄：自動儲存最近 15 筆，可星號標記常用
- 支援復原 / 重做（Undo / Redo）
- 所有資料儲存於本機（localStorage），不上傳任何內容
